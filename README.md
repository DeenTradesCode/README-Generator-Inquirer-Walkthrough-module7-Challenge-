
    Welcome to DTC README Generator!

## Description
This project is designed to use Node.js command line that dynamically generates a well professional README file based upon user input!

### Motivation
To apply what i learned using inquirer, node.js, to streamline a smooth proccess of creating high quality README files for developers.

### Problem Solved
It eliminates the need for developers to manually create README files for every new project, automating the process with ease.


### What I Learned
I learned how to use the Inquirer package to collect user input and dynamically generate a markdown file using Node.js.
